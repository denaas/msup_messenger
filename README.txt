CLIENT 
запустить файл Проект3.exe по пути msup_messenger/WindowsFormsClient/Debug. 
1) Ввести логин/ пароль. 
2) Зарегистрироваться, если данный пользователь ещё не зарегестрирован. 
3) Авторизироваться. 
4) В поле ввода сообщения ввести ИМЯ_ПОЛУЧАТЕЛЯ ТЕКСТ_СООБЩЕНИЯ, после чего нажать Enter, или кнопку отправить.

Крипто часть - .doc файл в корне.
